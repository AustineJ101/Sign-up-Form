# Sign-up-Form
This is a simple sign up form page built using HTML and CSS.  

This project was completed as part of The Odin Project Curriculum and mainly demonstrates my ability to: 
- Work with semantic form elements
- Implement input validation
- Style UI elements for an improved user experience

[Live Preview](https://austinej101.github.io/Sign-up-Form/)

# Features
- Different input types for password, text, email and tel
- Client side input validation for required inputs and phone number pattern
- Visual feedback when inputs are focused or invalid
